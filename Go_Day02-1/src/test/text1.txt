Упражнение 01: Подсчет предметов
Теперь мы можем найти наши файлы, но нам может понадобиться больше метаданных о том, что содержится в этих файлах. Давайте реализуем wc-подобную утилиту для сбора базовой статистики о наших файлах.
Прежде всего, давайте предположим, что наши файлы — это текстовые файлы в кодировке utf-8, поэтому ваш код должен работать и с текстом на русском языке (пока забудьте о специальных случаях, таких как арабский, нужны только английский и русский). Вы также можете игнорировать знаки препинания и использовать пробелы в качестве единственного разделителя слов.
Вам нужно будет реализовать три взаимоисключающих (одновременно можно указать только один, иначе вы получите сообщение об ошибке) флага для вашего кода: -l для подсчёта строк, -m для подсчёта символов и -w для подсчёта слов. Ваша программа должна работать следующим образом:

# Count words in file input.txt
~$ ./myWc -w input.txt
777 input.txt
# Count lines in files input2.txt and input3.txt
~$ ./myWc -l input2.txt input3.txt
42 input2.txt
53 input3.txt
# Count characters in files input4.txt, input5.txt and input6.txt
~$ ./myWc -m input4.txt input5.txt input6.txt
1337 input4.txt
2664 input5.txt
3991 input6.txt


Как вы можете видеть, ответом всегда является вычисленное число и имя файла, разделенные символом табуляции (\t). Если флаги не заданы, следует использовать -w поведение.
Важно: поскольку все файлы независимы друг от друга, для их одновременной обработки следует использовать горутины. Вы можете запустить столько горутин, сколько входных файлов указано для программы.
